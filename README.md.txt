# 📚 Projeto Livraria em Java

Este projeto tem como objetivo simular o funcionamento de uma **livraria**, sendo desenvolvido em **Java** como forma de prática e aprendizado.  
O sistema começará de forma simples e será **evoluído gradualmente** conforme novos conceitos forem aprendidos.

---

## 🚀 Funcionalidades previstas
- 📖 Cadastro de livros (título, autor, gênero, preço, etc.)
- 👤 Cadastro de clientes
- 🛒 Registro de compras e vendas
- 📊 Relatórios simples (livros cadastrados, vendas realizadas)
- 💾 Persistência de dados (arquivos e/ou banco de dados em versões futuras)

---

## 🛠️ Tecnologias utilizadas
- Java 17+ (ou versão que estiver utilizando)
- IDE: IntelliJ IDEA (ou outra de sua preferência)

---

## 🎯 Objetivos de aprendizado
O projeto é voltado para prática de:
- Estruturas de dados
- Orientação a Objetos (classes, herança, polimorfismo)
- Enumerações
- Laços e condições
- Manipulação de arquivos
- Integração futura com banco de dados e interface gráfica

---

## 📌 Status do projeto
🔨 Em desenvolvimento — primeiras funcionalidades estão sendo implementadas.
